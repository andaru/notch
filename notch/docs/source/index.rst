Notch: the Network Operator's Toolkit for Command-line Hacking
==============================================================

Notch is system for building network management applications. Notch
Agents expose a proxy interface to router and switch command-line
interfaces (CLI) for client tool use, as well as managing device
connections and authentication data.

See the Notch `project page`__ on Google Code for development information.
